## 第三部分：如何"养"OpenClaw？——让智脑多活几年的实战技巧

### 技巧1：模型轮换（避免单一超支）

**问题**：
- 只用Step，免费额度有限，超了就得付费
- 只用GPT-4o，钱包瞬间干瘪

**解决方案**：
- 不同任务用不同模型
- 设置自动fallback（超限自动切免费）

**我的配置表**：

| 任务类型 | 首选模型 | 备用模型 | 超限处理 |
|---------|---------|---------|---------|
| 日常对话 | Step-3.5 Flash | DeepSeek | 自动切 |
| 写文案 | Claude Haiku | Step | 自动切 |
| 图片分析 | GPT-4o | Claude 3.5 | 手动切换 |
| 深度思考 | GPT-4o | Claude 3.5 | 记录用量 |

**效果**：
- 没有模型会超额
- 成本稳稳50元/月
- 智脑没有饿肚子（不会因为欠费停摆）

---

### 技巧2：上下文管理（省tokens的核心）

**问题**：
- OpenClaw会把整个对话历史发给模型
- 历史越长，tokens越多，越贵

**例子**：
- 10轮对话后，历史可能3000+ tokens
- GPT-4o：0.02元/1k tokens → 单次0.06元
- 如果每天50次对话 → 3元/天 → 90元/月

**解决方案**：
- 每10轮对话 summarize 一次
- 只保留摘要，清空详细历史

**我的操作**：

```
# 对话进行到第10轮时，主动summarize
summary = """
对话摘要：
- 用户需求：写小红书文案（家装主题）
- 风格要求：爆款、带emoji、短句
- 已产出：3篇（阅读量：1W、3W、5W）
- 当前任务：写第4篇（厨房主题）
"""
# 清空详细历史，只保留这个摘要
```

**节省效果**：
- tokens 从 3000+ 降到 500-
- 单次成本降低 **80%**
- 每月省下 **70-80元**

---

### 技巧3：工具优先，别滥用模型

**问题**：
- 明明可以用 `web_search` 查天气，偏要让模型回答
- 明明可以 `read` 读文件，偏要让模型解析
- 习惯性把所有事都丢给模型 = 烧钱

**解决方案**：
- 能用内置工具，坚决不用模型
- 只把需要推理、创作的任务给模型

**我的清单**：

✅ **用工具（不耗模型tokens）：**
- `read` 读文件
- `web_search` 搜索
- `memory_search` 查记忆
- `exec` 运行命令
- `weather` 查天气

❌ **别用模型干这些（浪费）：**
- "今天天气怎么样？" → 应该用 `weather` 技能
- "帮我计算这个" → 应该用 `calc` 工具
- "文件里写了什么？" → 应该用 `read` 工具

**效果**：模型调用次数减少 **40%**，月省 **20元**

---

### 技巧4：批次处理（减少对话轮次）

**问题**：
- 用户一次要10个标题，却问10次
- 每次都要新建对话，重新加载上下文
- 成本 ×10

**解决方案**：
- 一次性批量提交
- 明确告诉模型："我要10个标题，一次性生成"

**对比**：

```
低效做法：
用户：帮我写3个标题 → 调用1次
用户：再写3个 → 调用2次
用户：再写4个 → 调用3次
总计：3次调用

高效做法：
用户：我要10个标题，一次性给我 → 调用1次
总计：1次调用

节省：2次 = 省60%成本
```

**效果**：批量处理可减少 **30-50%** 调用次数

---

### 技巧5：监控与预警（防止破产）

**问题**：
- 不知不觉用超了额度
- 月底收到账单吓一跳：**怎么花了500元？！**

**解决方案**：
- 设置预算上限
- 每周检查用量
- 自动降级策略

**我的监控配置**：

```
月度预算（50元）：
- Claude Haiku：30元（200次）⚠️ 核心风险
- GPT-4o：15元（10次）
- 备用：5元

每周检查（周日下午）：
1. Claude用量是否 > 80%？
   → 是：自动切Step，提醒"需要充值吗？"
2. GPT-4o是否已用10次？
   → 是：封存GPT，改用Claude 3.5
3. Step免费额度是否剩余？
   → 低于10次： DeepSeek

预警机制：
- 周用量 > 70% → 提醒"本周超支预警"
- 月用量 > 90% → 暂停所有付费任务
```

**效果**：
- 没有意外超支
- 智脑稳如老狗
- 钱包安全
