# AI Skills

This repository contains a collection of AI skills for OpenClaw agents.

## Skills List

### 1. duckduckgo-search
- **Description**: DuckDuckGo search integration
- **Files**: SKILL.md, search.py

### 2. image-analysis
- **Description**: Image analysis capabilities
- **Files**: SKILL.md, analyze.py

### 3. image-generate
- **Description**: Image generation using Python scripts
- **Files**: SKILL.md, scripts/image_generate.py

### 4. veadk-go-skills
- **Description**: VeADK-Go related skills and tools
- **Files**: SKILL.md, common/*.md, converter/*.md

### 5. veadk-skills
- **Description**: VeADK related skills and tools
- **Files**: SKILL.md, references/*.md, scripts/*.py

### 6. video-generate
- **Description**: Video generation using Python scripts
- **Files**: SKILL.md, scripts/video_generate.py

## Usage

These skills are designed to work with OpenClaw agents. Place them in your `~/.openclaw/workspace/skills/` directory.

## Note

- Personal identity files (IDENTITY.md, USER.md) are excluded from this repository
- Virtual environments and Python cache files are also excluded
