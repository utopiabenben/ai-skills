# AI Skills

This repository contains a collection of AI skills for OpenClaw agents.

## Skills List

### 本地开发技能 (Local Skills - 13个)

1. **batch-renamer** - 批量文件重命名工具
2. **photo-organizer** - 照片批量整理工具
3. **download-organizer** - 下载文件自动分类工具
4. **video-organizer** - 视频文件批量重命名和整理工具
5. **music-tagger** - 音乐文件批量标签工具
6. **file-sorter** - 通用文件智能分类工具
7. **doc-converter** - 文档格式批量转换工具
8. **duckduckgo-search** - DuckDuckGo搜索集成
9. **image-analysis** - 图片分析与OCR技能
10. **image-generate** - 图片生成功能
11. **veadk-go-skills** - VeADK-Go相关技能和工具
12. **veadk-skills** - VeADK相关技能和工具
13. **video-generate** - 视频生成功能

### 系统内置技能 (Built-in Skills - 6个)

1. **weather** - 天气查询
2. **healthcheck** - 主机安全加固
3. **nano-pdf** - PDF编辑
4. **github** - GitHub操作
5. **notion** - Notion页面管理
6. **obsidian** - Obsidian笔记库操作

### 第三方安装技能 (Clawhub Skills - 2个)

1. **obsidian-ontology-sync** - Obsidian笔记与结构化本体同步
2. **openclaw-tavily-search** - Tavily网络搜索

**总计：21个技能** 🎉

## Detailed Inventory

See `SKILLS_INVENTORY.md` for comprehensive skills inventory and `SKILLS_CONSISTENCY_REPORT.md` for consistency check report.

## Usage

These skills are designed to work with OpenClaw agents. Place them in your `~/.openclaw/workspace/skills/` directory.

## Note

- Personal identity files (IDENTITY.md, USER.md) are excluded from this repository
- Virtual environments and Python cache files are also excluded
